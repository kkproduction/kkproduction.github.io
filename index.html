Текстовые редакторы — твои рабочие инструменты

Текстовые редакторы — твои рабочие инструменты Поскольку верстка — это работа с текстовыми документами, ее можно осуществить даже в обыкновенном блокноте, который поставляется вместе с Windows. Однако, как для более деликатной работы необходимы более точные инструменты, так для написания хорошего сайта просто необходим качественный текстовый редактор. Самый простой выбор для студента — Notepad++. Он просто подсвечивает синтаксис и не перегружен функциями. Чуть сложнее Sublime Text 3 с установленным плагином для работы с веб дизайном. Функций по более, оформление поприятнее, есть функции вроде авто закрытия тега. Самым профессиональным для этих целей является JetBrains WebStorm. Он напомнит тебе о не закрытом теге, отформатирует тебе текст и вообще сделает тебя готовым ко всем неожиданностям.

Теоретические материалы

Даже имея самые передовые инструменты ты сядешь в лужу, если не знаешь, как их применить. Необходимый минимум дают на наших лекциях, но минимум это слишком мало для чего-то хорошего. А делать не хорошее — не хорошо. Так что дополнительные материалы тебе в помощь. На каждом из перечисленных ниже сайтов есть как гайды по созданию сайта с нуля, так и библиотеки со всеми тегами и описанием их работы. Считаю, что этого вполне достаточно для создания чего-то выше среднего.
